# musicstore_SQL
🎵 Music Store Data Analysis (SQL)

About the Project

This project is based on analyzing a music store database using SQL.
I worked on solving different business-related questions using queries.

---

Tools Used

- SQL (MySQL)

---

Dataset Overview

The database includes:

- Customers
- Invoices
- Tracks
- Artists
- Genres

---

What I Did

- Wrote SQL queries using JOINs, GROUP BY, and aggregations
- Analyzed which genres generate the most revenue
- Identified top customers and their spending behavior
- Explored sales performance across different countries

---

Key Findings

- A small group of customers contributes a large share of revenue
- Some music genres are consistently more popular than others
- Repeat customers are very important for business growth
